Hide Deprecated Applications=Ukryj przestarzałe aplikacje
This setting hides deprecated applications from the list of available apps Deprecated apps will still however display within the installed and previous apps sections=To ustawienie ukrywa przestarzałe aplikacje na liście dostępnych aplikacji. Przestarzałe aplikacje będą jednak nadal wyświetlane na liście zainstalowanych oraz poprzednich aplikacji.
Hide Incompatible Applications=Ukryj niekompatybilne aplikacje
This setting will hide applications that are listed as being incompatible with your version of Unraid=To ustawienie ukrywa aplikacje które nie są kompatybilne z Twoją wersją Unraida
Enable Reinstall Default=Włącz reinstalację domyślną
This option enables another button on the cards and popups which will allow you to reinstall an installed application using its default template values This option can be confusing at times, and is only generally used if you wish to install and have an application running twice simultaneously=To ustawienie włącza dodatkowy przycisk na karcie i w wyskakujących okienkach. Pozwala on na ponowną instalację zainstalowanej już aplikacji, używając jej domyślnych ustawień. Ta opcja może wydawać się czasem myląca, jej głównym zastosowaniem jest sytuacja gdy chcesz mieć tę samą aplikację uruchomioną dwukrotnie w tym samym czasie
Enable additional search results from dockerHub=Włącz dodatkowe przeszukiwanie wyników z dockerHub
This setting enables CA to retrieve additional search results from dockerHub Note that docker must be enabled for this to be operational=To ustawienie umożliwia CA przeszukiwanie serwisu dockerHub. Aby to działało, docker musi być włączony w ustawieniach Unraid
Allow CA to send any emergency notifications=Zezwól CA na wysyłanie powiadomień w wyjątkowych sytuacjach
This setting allows CA to periodically scan your system and alert you to the presence of=To ustawienie zezwala CA na koresowe skanowanie Twojego systemu i powiadamianie Cię o obecności
